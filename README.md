# Inkscape Tile Maker

Create SVG-based tilesets for use in games, using Inkscape.

## Features
- Configure tiles for automatic export, saved into the SVG itself
- Live preview of the tileset, showing how the tiles will look when exported
- Paint mode to help preview the tiles in-context
- Export "Materials" or tiles with the same subject (e.g. grass)
- Export tilesets directly into the Unity Engine
- Support for Dual Grid tiles.
- Structural support for Rule Tiles (not yet implemented)
- Slice and export a tileset into individual tiles (svg or png)


Feel free to suggest new features or improvements!

At the moment, I only add things that I need for my own projects, but I will try to add features that are requested by others as well.

## Installation

1. Go to the latest release. 
2. Pick the desired method (Installer, Prebuilt ZIP, or Compile yourself from source code).
3. Run the program.

## Creating a New SVG Tileset from Templates

Currently, the best way to start is from a template. From the landing page, you'll see the options within Create New.
You can also open an existing SVG file if it originated from Inkscape.

## Windows Only
The app is configured for Windows only, but it is built on .NET MAUI, so multiplatform is possible if there is interest.
